# MakerCommons — `.github`

This is the **community & organization repo** for
[MakerCommons](https://github.com/MakerCommons). It holds the things that apply
to the whole collective:

| File | What it is |
|------|------------|
| [`MANIFESTO.md`](MANIFESTO.md) | Why we exist and what we believe. Start here. |
| [`profile/README.md`](profile/README.md) | The public landing page shown on the org's GitHub profile. |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | How to bring a problem, build a tool, or help out. |
| [`GOVERNANCE.md`](GOVERNANCE.md) | How the collective makes decisions. |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | How we treat each other. |
| [`SECURITY.md`](SECURITY.md) | How to report a vulnerability. |
| [`TOOLS.md`](TOOLS.md) | Index of everything the collective has shipped. |
| [`.github/ISSUE_TEMPLATE/`](.github/ISSUE_TEMPLATE) | Templates for bringing a problem or proposing a tool. |

Because this repo is named `.github`, GitHub uses it as the **default source**
for community health files across every MakerCommons repository. A tool repo
that doesn't ship its own `CONTRIBUTING.md` or issue templates automatically
inherits the ones here.
